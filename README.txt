2) Reconfig port and server address on files: /andromeda/server/config.conf or /andromeda/client/config.con
3) Start server: run command "server_dev_b_plant"
4) On client:
	Encrpt file: client_dev_b_plant Encrypt <path of file need encrypt> <path of result file>
	Decrpt file: client_dev_b_plant Decrypt <path of file need decrypt> <path of result file>
